这个是 Menu 的 defaultOpenKeys 控制的。
位置：
[AppShell.tsx](/Users/huagu/IDEA/yuheng/apps/web/src/components/AppShell.tsx)
现在应该是类似这样：
<Menu
  theme="dark"
  mode="inline"
  defaultOpenKeys={['group-one', 'group-two', 'group-three']}
  selectedKeys={selectedKeys}
  items={antdMenuItems}
  onClick={({ key }) => openMenu(String(key))}
/>
如果你想项目第一次打开时菜单都关闭，就改成：
defaultOpenKeys={[]}
或者直接去掉：
<Menu
  theme="dark"
  mode="inline"
  selectedKeys={selectedKeys}
  items={antdMenuItems}
  onClick={({ key }) => openMenu(String(key))}
/>
注意：
defaultOpenKeys 只控制首次渲染默认展开哪些一级菜单
group-one / group-two / group-three 就是一级菜单的 key
如果想后续受状态控制，要用 openKeys + onOpenChange
如果只是“第一次打开默认都收起”，改 defaultOpenKeys={[]} 就够了
