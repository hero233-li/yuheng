在“数据维护”页面里，搜索项的**换行**主要由两个地方控制：

1. 每个字段的 `span`
   
文件：

[SearchForm2Page.tsx](/Users/huagu/IDEA/yuheng/apps/web/src/apps/web/SearchForm2Page.tsx)

动态字段这里：

```tsx
<Col span={field.span} key={field.name}>
```

`span` 是 Ant Design 的 24 栅格：

```text
span: 24  一整行
span: 12  半行
span: 8   一行三个
span: 6   一行四个
```

也就是说，一行累计超过 24，就会自动换到下一行。

2. `Row gutter`

同一个文件里：

```tsx
<Row gutter={16}>
```

`gutter` 控制字段之间的间距，不直接控制换行，但会影响视觉距离。

---

搜索按钮的**居中**是这个 class 控制的：

文件：

[SearchForm2Page.tsx](/Users/huagu/IDEA/yuheng/apps/web/src/apps/web/SearchForm2Page.tsx)

```tsx
<Space className="form-action-center">
```

对应样式在：

[styles.css](/Users/huagu/IDEA/yuheng/apps/web/src/styles.css)

```css
.form-action-center {
  display: flex;
  justify-content: center;
  width: 100%;
}
```

所以：

- 字段换行：改字段 `span`
- 按钮居中：改 `.form-action-center`
- 字段间距：改 `Row gutter={16}`
