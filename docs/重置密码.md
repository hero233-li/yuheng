可以用两种方式。

**方式一：Row 居中**

在表单外层 `Row` 上加：

```tsx
<Row gutter={16} justify="center">
```

比如重置密码只有两个字段：

```tsx
<Row gutter={16} justify="center">
  <Col span={6}>...</Col>
  <Col span={6}>...</Col>
</Row>
```

这样两个 `Col` 会整体居中。

**方式二：给 Row 加 class**

```tsx
<Row gutter={16} className="form-row-center">
```

CSS：

```css
.form-row-center {
  justify-content: center;
}
```

如果你还想字段宽一点，就调 `span`：

```tsx
<Col span={8}>...</Col>
<Col span={8}>...</Col>
```

核心就是：

- 字段宽度：`Col span`
- 整行居中：`Row justify="center"` 或 CSS `justify-content: center`
