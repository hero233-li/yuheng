现在产品申请里的几个 switch：

- 白名单 `whitelist`
- 红盾 `redShield`
- 征信 `creditReport`
- 法人 `legalPerson`

本质上是放在公共字段 `commonFields` 里的，所以默认是：**所有产品都能使用**。

位置在：

[searchFormConfig.ts](/Users/huagu/IDEA/yuheng/apps/web/src/apps/web/searchFormConfig.ts)

```ts
const commonFields: FieldConfig[] = [
  ...
  {
    name: 'whitelist',
    label: '白名单',
    type: 'switch',
    ...
  },
  {
    name: 'redShield',
    label: '红盾',
    type: 'switch',
    ...
  },
  {
    name: 'creditReport',
    label: '征信',
    type: 'switch',
    ...
  },
  {
    name: 'legalPerson',
    label: '法人',
    type: 'switch',
    ...
  },
];
```

它们和产品的关系是在这里合并的：

```ts
const commonWithProductRules = commonFields.map((field) =>
  applyDynamicFieldRules(
    {
      ...field,
      ...(product.fieldOverrides?.[field.name] || {}),
    },
    values,
  )
);
```

意思是：

1. 所有产品默认都有这些公共字段。
2. 如果某个产品在 `fieldOverrides` 里配置了同名字段，就覆盖公共字段的配置。
3. 可以按产品控制：
   - 是否显示 `visible`
   - 是否提交 `submit`
   - 是否可编辑 `editable`
   - 默认值 `defaultValue`
   - 宽度 `span`
   - 文案 `checkedLabel / uncheckedLabel`

比如产品B不需要征信：

```ts
fieldOverrides: {
  creditReport: {
    visible: false,
    submit: false,
  },
}
```

比如产品A白名单默认关闭：

```ts
fieldOverrides: {
  whitelist: {
    defaultValue: false,
  },
}
```

所以结论是：

**默认所有产品都能用，但可以通过每个产品自己的 `fieldOverrides` 控制某个 switch 是否可用。**



可以在表单渲染组件里统一加 `size="large"`。

比如产品申请在：

[JobCreatePage.tsx](/Users/huagu/IDEA/yuheng/apps/web/src/apps/web/JobCreatePage.tsx)

`renderField` 里改：

```tsx
if (field.type === 'select') {
  return (
    <Select
      {...commonProps}
      size="large"
      ...
    />
  );
}

return <Input {...commonProps} size="large" />;
```

如果 switch 也想变大：

```tsx
<Switch size="default" />
```

Ant Design 的 `Switch` 只有 `small` 和 `default`，没有 `large`。

如果是“数据维护”页面，也是在它自己的字段渲染方法里加：

[SearchForm2Page.tsx](/Users/huagu/IDEA/yuheng/apps/web/src/apps/web/SearchForm2Page.tsx)

```tsx
<Select size="large" />
<Input size="large" />
```

如果想全局统一，可以在 `ConfigProvider` 里设置：

```tsx
<ConfigProvider >
  ...
</ConfigProvider>
```

这个会影响所有 Ant Design 表单控件，不只是搜索表单。
